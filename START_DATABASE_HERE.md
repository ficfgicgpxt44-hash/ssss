### "لا أستطيع الدخول"
```
1. تحقق من الإيميل: ficfgicgpxt44@gmail.com
2. افتح في نافذة بدون الوضع الخاص
3. امسح ذاكرة المتصفح (Ctrl+Shift+Delete)

```

### "الحالة لم تظهر بعد الإضافة"
```
1. انتظر ثانية أو اثنتين
2. احدّث الصفحة (F5)
3. تحقق من الاتصال بالإنترنت
4. انقر "Sync" في لوحة التحكم
```

### "الصور لا تحمل"
```
1. تأكد من حجم الصورة < 2MB
2. استخدم JPG بدلاً من PNG
3. تأكد من الاتصال بالإنترنت
4. حاول صورة أخرى
```

### "الصور بطيئة جداً"
```
1. استخدم صور أصغر حجماً
2. ضغط الصور قبل الرفع
3. استخدم صيغة JPG
4. تحقق من سرعة الإنترنت
```
