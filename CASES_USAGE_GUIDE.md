# 📘 دليل استخدام نظام الحالات الطبية

## الخطوات السريعة (3 خطوات فقط)

### ✅ الخطوة 1: تسجيل الدخول للإدارة
```
1. ادخل الموقع
2. ابحث عن منطقة مخفية في معرض الحالات
3. انقر عليها
4. ستفتح نافذة تسجيل دخول
5. اختر "Google" للدخول
6. استخدم حسابك: ficfgicgpxt44@gmail.com
```

### ✅ الخطوة 2: إضافة حالة جديدة
```
1. بعد التسجيل الدخول، ستظهر لوحة التحكم (Admin Dashboard)
2. انقر على زر "إضافة حالة جديدة" (+ Add Case)
3. امل البيانات:

   📝 العنوان (Title):
   مثال: "حالة أسنان علوية مع ترميم"
   
   📂 الفئة (Category):
   اختر من:
   - Endodontics (معالجة الجذور)
   - Prosthodontics (التركيبات والتلبيسات)
   - Surgery (الجراحة)
   - Pedodontics (طب أسنان الأطفال)
   - Cosmetic Fillings (الحشوات التجميلية)
   
   📄 الوصف (Description):
   مثال: "حالة تتضمن ترميم الأسنان العلوية مع استخدام حشوات دائمة"
   
   🖼️ الصور (Images):
   - انقر "أضف صور"
   - اختر صور من الكمبيوتر (قبل، أثناء، بعد)
   - يمكن أضافة حد أقصى 20 صورة
   - الصور تحفظ تلقائياً

4. انقر "حفظ الحالة" (Save Case)
5. انتظر حتى ترى رسالة النجاح
```

### ✅ الخطوة 3: الحالات ستظهر تلقائياً للزوار
```
النظام سيقوم تلقائياً بـ:
✓ حفظ الحالة في قاعدة البيانات (Firestore)
✓ تحديث المعرض
✓ إظهار الحالة لجميع الزوار
✓ تنظيمها حسب الفئة
```

## 📸 نصائح حول الصور

### جودة الصور:
- 📐 الحجم المثالي: 1200x800 بكسل
- 📁 الصيغة: JPG (أسرع) أو PNG
- 💾 الحجم: أقل من 2MB لكل صورة
- 🎨 الجودة: واضحة وحادة

### أنواع الصور:
```
يفضل إضافة صور تسلسلية:
1. صورة أولية (قبل العلاج)
2. صور أثناء العمل
3. صورة نهائية (بعد العلاج)
4. صور زوايا مختلفة
```

### نصائح عملية:
```
✓ صور الأسنان من زوايا مختلفة
✓ تغطية مناطق الفم المختلفة
✓ تصوير الجانب الأيمن والأيسر
✓ تصوير من الأمام والخلف
✓ عدم وضع وجه المريض
```

## 🐛 حل المشاكل الشائعة

### ❌ المشكلة: لا أستطيع الدخول

**الحل:**
```
1. تأكد من استخدام الإيميل الصحيح: ficfgicgpxt44@gmail.com
2. اعادة تحميل الصفحة (F5)
3. امسح ذاكرة المتصفح (Cache)
4. حاول من متصفح آخر
5. تحقق من تفعيل Google Login في Firebase
```

### ❌ المشكلة: لا تظهر الحالات بعد الإضافة

**الحل:**
```
1. انتظر ثانية أو اثنتين
2. احدّث الصفحة (F5)
3. تحقق من الاتصال بالإنترنت
4. انقر "مزامنة" (Sync) في Dashboard
5. تحقق من Firebase Console
```

### ❌ المشكلة: الصور لا تحمل

**الحل:**
```
1. تحقق من حجم الصورة (أقل من 2MB)
2. استخدم صيغة JPG بدلاً من PNG
3. تأكد من جودة الإنترنت
4. احذف الصورة وأضفها مرة أخرى
```

### ❌ المشكلة: الحالة المحذوفة عادت

**الحل:**
```
1. قد يكون هناك تأخير في المزامنة
2. انتظر دقيقة واحدة
3. احدّث الصفحة (F5)
4. اذهب إلى Firebase Console للتحقق
```
